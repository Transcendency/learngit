## create a repository

1. mkdir learngit
2. cd learngit
3. git init


Git is a version control system.
Git is free software.

4. git add [tells git to add the file to repository]
5. git commit -m "wrote a read me file" [tells git to submit the file to repository]

why put file to git needs "add" and "commit"? Because commit can submit multiple files at a time, so you can add multiple times.

for example
git add file1.txt
git add file2.txt file3.txt
git commit -m "add 3 files"
